FEATURES
// MECHANICS
☗
DOMAIN EXPANSIONS
Three unique domains — Unlimited Void, Malevolent Shrine (open domain), Chimera Shadow Garden. Each with full cinematic animation and battle effects.
⚙
MAHORAGA ADAPTATION
Divine shikigami that adapts to every attack. After 4 hits, achieves 100% adaptation — immune and heals. Nerfed for balance: 1 adaptation per turn max.
∞
INFINITY PASSIVE
Gojo's Infinity reduces all incoming damage. Mahoraga's Adaptive Cleave gradually pierces it. Sukuna's Red can bypass it completely.
🐕
SHIKIGAMI SYSTEM
Megumi can summon Divine Dogs, Nue, Great Serpent, and Giant Toad — each with unique effects. Chimera Shadow Garden unlocks all simultaneously.
🎵
PROCEDURAL AUDIO
Full Web Audio API sound design. Unique sounds per character, per attack, per shikigami. Three ambient BGM tracks that shift with battle intensity.
🖼
SCENARIO SYSTEM
Three battle backgrounds loaded randomly each fight. Domain images appear AFTER their activation animation ends. Add your own: drop PNGs in /images/scenarios/.

HOW TO ADD A NEW CHARACTER

Copy one of the files in chars/ as a template and fill in the stats. Then register it in the main game file:

// 1. Create chars/nanami.js using gojo.js as template // 2. In jjk_sorcerers_duel_v9.html, find const DATA = { ... } // and add your character: const DATA = { gojo: { /* ... */ }, sukuna: { /* ... */ }, megumi: { /* ... */ }, nanami: CHAR_NANAMI, // ← add this line }; // 3. Add portrait: images/chars/nanami.png // 4. Optionally add select sound in SoundEngine._selectNanami()
